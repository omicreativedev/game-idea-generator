# Game Idea Generator

A simple web application that generates random game ideas by combining adjectives and character types.

## Features

- Generates random game ideas based on a "fill in the blank" template.
- Click on any term to freeze/unfreeze it.
- Frozen terms remain unchanged when generating new ideas.
- Responsive design that works on mobile and desktop.

## How to Use

1. Visit the [GitHub Pages site](https://your-username.github.io/repository-name)
2. Click "Generate New Idea" to create a random game concept
3. Click on any generated term to freeze it (it won't change on future generations)
4. Click "Reset All" to unfreeze all terms and generate a completely new idea

## File Structure

- `index.html` - Main HTML file
- `style.css` - Styling for the webpage
- `script.js` - JavaScript functionality
- `character_descriptions.txt` - List of adjectives
- `characters.txt` - List of character types

## Customization

You can easily customize the generator by editing the text files:

1. Add more adjectives to `character_descriptions.txt` (one per line)
2. Add more character types to `characters.txt` (one per line)

## Deployment to GitHub Pages

1. Create a new repository on GitHub or Fork
2. Go to repository Settings > Pages
3. Under "Source", select "Deploy from a branch"
4. Select the branch (usually `main` or `master`) and folder (`/root`)
5. Click Save
6. Your site will be published at `https://your-username.github.io/repository-name`
